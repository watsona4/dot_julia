## Usage

Just include the `Bioinformatics.jl` file with

```julia
include("Bioinformatics.jl")
```

or install the library with

```julia
(v1.1) pkg> add "https://github.com/mrtkp9993/Bioinformatics.jl"
```

and you're ready to use!
